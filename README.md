# Pharmacy Management & Online Ordering System

## Overview
This project is a **web-based pharmacy management system** designed to streamline traditional pharmacy operations. It replaces manual processes with an automated solution for **inventory management, customer record handling, and online ordering**.

## Features
- **Inventory Management**: Track and manage medicines efficiently.
- **Customer Records**: Maintain detailed and secure customer information.
- **Online Ordering**: Allow customers to place orders through a secure web interface.
- **Secure Authentication**: Session-based login for authorized personnel.

## Technologies Used
- **Backend**: Java EE (Jakarta EE)
- **Frontend**: JSP, CSS
- **Database**: MySQL
- **Server**: GlassFish 7

## Problem
Traditional pharmacy systems relied on manual processes, resulting in:
- Inefficient inventory tracking
- Poor customer record management
- Lack of online ordering capabilities

## Solution
The web application addresses these issues by:
- Automating inventory monitoring
- Structuring and securing customer data management
- Providing a secure online ordering platform for customers

## Result
- **Operational Efficiency**: Streamlined pharmacy workflows
- **Data Management**: Improved accuracy and accessibility of customer and inventory data
- **Enhanced Customer Experience**: Secure and convenient online ordering
